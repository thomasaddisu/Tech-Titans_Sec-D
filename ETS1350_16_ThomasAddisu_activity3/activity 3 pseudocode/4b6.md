# Pseudocode: Hollow inverted Half Pyramid

1. Start the program
2. Declare integer variable `row`
3. Read `row`
4. For `j = 0` to `j < row`:\
    a. For `i = row - 1` to `i >= j`:
        - If `j == 0`:
            - Print star\
        - Else if `i == row - 1` or `i == j`:
            - Print star\
        - Else:
            - Print a space\
    b. Print a new line after each iteration of the outer loop
5. End the program
