# Pseudocode: 

1. Start the program
2. Declare integer variable `row`
3. Read `row`
4. For `i = 0` to `i < row`:\
    a. For `j = row - 1` to `j > i`:
        - Print two spaces\
    b. For `z = i` to `z >= 0`:
        - Print `z + 1` followed by a space\
    c. Print a new line after each iteration of the outer loop
5. End the program
