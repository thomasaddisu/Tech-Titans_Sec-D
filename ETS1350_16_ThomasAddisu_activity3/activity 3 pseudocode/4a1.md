# question 
Print numbers 1 to column count, row times



# Pseudocode: 

1. Start the program
2. Declare integer variables `row`, `column`
3. Read  `row` and `column`
4. For `j = 0` to `j < row`:\
    a. For `i = 0` to `i < column`:
        - Print `i + 1` followed by a space\
    b. Print a new line after each inner loop
5. End the program
