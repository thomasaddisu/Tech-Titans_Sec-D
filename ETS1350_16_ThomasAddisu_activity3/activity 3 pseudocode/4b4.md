# Pseudocode: hollow quadrilateral

1. Start the program
2. Declare integer variables `row`, `column`
3. Read `row` and `column`
4. For `i = 0` to `i < row`:\
    a. For `j = 0` to `j < column`:\
        - If `i == 0` or `i == row - 1`:
            - Print "* "\
        - Else if `j == 0` or `j == column - 1`:
            - Print "* "\
        - Else:
            - Print two spaces\
    b. Print a new line after each iteration of the outer loop
5. End the program
